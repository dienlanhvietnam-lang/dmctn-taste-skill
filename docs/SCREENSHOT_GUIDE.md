# Screenshot Guide — DMCTN Taste Skill

Real UI only — no AI mockups labeled as product shots.

## Files to create (`store-assets/`)

| File | Tab / subject |
|------|----------------|
| `dashboard-overview.png` | Overview — Full status, install buttons |
| `install-skills.png` | Install **or** Skills (13 skills) |
| `prompt-output.png` | Prompt mẫu — filled Output textarea |
| `coding-result.png` | Agent Design Read / redesigned page (e.g. index.html) |

## Crop rules

- Crop to **dashboard webview** or **browser result** — not full IDE chrome.
- Target ~1280×720 (16:9), PNG, &lt; 500 KB each when possible.
- No secrets, tokens, or private paths in frame.

## Steps

1. Open extension root → `npm install` → **F5**.
2. Temp workspace → **Install Full** → status **13/13**.
3. **DMCTN Taste: Open Dashboard** → capture Overview, Install/Skills, Prompt mẫu.
4. Run agent on UI task → capture coding result separately for `coding-result.png`.

## After capture

- Update `store-assets/README.md` status table to **Present**.
- Reference images in root `README.md` (paths work on GitHub when pushed).
- Upload to Marketplace / Open VSX listing UI.
