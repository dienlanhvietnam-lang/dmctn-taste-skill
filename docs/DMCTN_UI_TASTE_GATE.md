# DMCTN UI Taste Gate

## Mốc 1: Audit

- Xem UI hiện tại
- Ghi màn hình/route chính
- Ghi lỗi thị giác/UX/mobile
- Không sửa code ở mốc này nếu task là audit

## Mốc 2: Design Read

Trả lời:

```text
Đọc brief này là: ...
```

## Mốc 3: Direction

Chọn:

- màu
- font
- layout
- component density
- motion level
- điểm nhớ thị giác

## Mốc 4: Code

Chỉ code sau khi Gate PASS.

## Mốc 5: Verify

- build/test
- mobile check
- visual regression thủ công nếu có ảnh
- SEO/security nếu liên quan

## Tiêu chí PASS

- không giống template AI
- người dùng thật dùng được
- mobile không vỡ
- trạng thái đầy đủ
- không phá chức năng cũ
- có evidence test
